Introduction

Santa's Sweatshop was a 72-hour programming project I undertook for the Winter Melon Jam 2023. This involved planning the concept for the game, programming the logic and drawing the pixel art assets.

Results

This game won Judges' Choice in the Novice section of the competition.

The only assets not made by me nor within the time limit are:

  • The song [It's Christmas! by KPM]
  • The SFX
  • The font (Courier New)
